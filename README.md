# AI and ML in Architecture Design

Welcome to the official repository for **AI and ML in Architecture Design**. This repository serves as a centralized hub for all course materials, including hands-on Jupyter Notebooks and reference PDFs.

---

## 📌 Course Overview
This course explores the intersection of artificial intelligence, machine learning, and architectural design. Students will learn how Machine Learning algorithms, Deep Learning, Reinforcement Learning and other data-driven methods can be applied to spatial analysis, structural optimization, conceptual design, and urban planning.
